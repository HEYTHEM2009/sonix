# Sonix — منصة تواصل اجتماعي كاملة

## نظرة عامة

Sonix هو تطبيق تواصل اجتماعي كامل مبني بـ **Laravel 13** (خادم API) و **React Native / Expo SDK 57** (تطبيق الموبايل).

---

## المميزات

### المصادقة
- تسجيل حساب جديد
- تسجيل الدخول
- إعادة تعيين كلمة المرور
- تغيير كلمة المرور
- حذف الحساب
- المصادقة الثنائية (2FA)

### المنشورات
- إنشاء منشورات (نص/صورة/فيديو)
- تعديل/حذف المنشورات
- إعجاب/إلغاء الإعجاب
- التعليقات
- الحفظ (Bookmarks)
- المشاركة (Native Share)
- الوسوم (#hashtag)
- الم提及ات (@mention)
- إحصائيات المنشورات
- تثبيت/إلغاء تثبيت المنشورات

### القصص
- إنشاء قصص (صورة/فيديو/نص)
- عرض القصص
- تفاعل القصص
- إحصائيات القصص
- أرشيف القصص (Highlights)
- مشاركة القصص

### الرسائل
- رسائل فردية (1:1)
- رسائل صوتية
- رسائل صورة
- تفاعل الرسائل (إيموجي)
- مؤشر الكتابة
- حالة القراءة
- وضع الاختفاء (Vanish Mode)
- توجيه الرسائل
- كتم المحادثات
- تثبيت المحادثات
- حذف المحادثات

### دردشة المجموعات
- إنشاء مجموعات
- إضافة/حذف أعضاء
- رسائل المجموعات

### الاستكشاف
- صفحة الاستكشاف
- المنشورات الرائجة
- المستخدمون المقترحون
- البحث عن المستخدمين
- سجل البحث

### السلامة
- حظر المستخدمين
- الإبلاغ عن المحتوى
- فلتر الكلمات البذيئة
- حماية ضد السحب
- رؤوس الأمان

### مميزات إضافية
- زوار الملف الشخصي
- شارات المستخدم
- قوالب الملف الشخصي
- فيديوهات Reels
- دعم RTL (العربية)
- لغتان (إنجليزية/عربية)

## التثبيت

### الخادم
```bash
cd laravel-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

### تطبيق الموبايل
```bash
cd expo-app
npm install
cp .env.example .env
npx expo start
```
