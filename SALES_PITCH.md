## What is Sonix?

Sonix is a social media app I built from scratch over several months. It combines features from Instagram, Snapchat, WhatsApp, and TikTok into one platform. The backend is Laravel 13 with PostgreSQL, and the mobile app is React Native with Expo SDK 57.

What makes it different: full Arabic/RTL support, voice messages with waveform animation, vanish mode (auto-delete messages), and group chat. These features together are hard to find in open-source projects.

- **Authentication:** Register, login, password reset, 2FA
- **Posts:** Text, image, video, likes, comments, bookmarks, hashtags, mentions

- **Messaging:** 1:1 DMs, voice messages, image messages, reactions, typing indicators, read receipts, vanish mode, forward, mute/pin
