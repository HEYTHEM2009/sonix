# Sonix — منصة تواصل اجتماعي كاملة

## نظرة عامة

Sonix هو تطبيق تواصل اجتماعي كامل جاهز للتشغيل، مبني بأحدث تقنيات الويب. يشمل تطبيق موبايل (React Native/Expo) وخادم API (Laravel) وقاعدة بيانات (PostgreSQL).
