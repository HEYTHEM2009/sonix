# 🎵 Sonix — منصة تواصل اجتماعي كاملة للبيع

## ملخص المشروع

**Sonix** هو تطبيق تواصل اجتماعي كامل جاهز للتشغيل، مبني بأحدث التقنيات. يشمل تطبيق موبايل وخادم API وقاعدة بيانات.

## التقنيات المستخدمة

### الخادم (Backend)
- **اللغة:** PHP 8.3
- **الإطار:** Laravel 13.8
- **قاعدة البيانات:** PostgreSQL 15+
- **المصادقة:** Laravel Sanctum 4.3
- **التخزين:** Cloudinary
- **القائمة/الكاش:** Redis
- **الوقت الحقيقي:** Laravel Reverb
- **الخادم:** PHP-FPM + Nginx
- **الحاوية:** Docker

### تطبيق الموبايل (Frontend)
- **الإطار:** React Native 0.86
- **SDK:** Expo 57
- **التنقل:** React Navigation 7
- **الحركات:** React Native Reanimated 4.5
- **HTTP Client:** Axios 1.7
- **الكاميرا:** expo-camera 57
- **الصوت:** expo-audio 57
- **الفيديو:** expo-video 57

## بدء التشغيل

```bash
# الخادم
cd laravel-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve

# تطبيق الموبايل
cd expo-app
npm install
cp .env.example .env
npx expo start
```

## لماذا Sonix؟

1. **جاهز للتشغيل** — لا تحتاج لسنة تطوير
2. **تقنيات حديثة** — Laravel 13 + Expo SDK 57
3. **دعم RTL** — العربية + الإنجليزية بالكامل
4. **مميزات فريدة** — رسائل صوتية، وضع الاختفاء، زوار الملف الشخصي
5. **توثيق شامل** — أي مطور يمكنه التشغيل
6. **Docker** — نشر سهل وسريع
